¿Que es esto?

Una aplicación web que permite gestionar el sistema de alquiler de bicicletas en la ciudad. Las bicicletas se toman y devuelven en diferentes estaciones, que están registradas en el sistema.

Cada estación tiene una serie de datos asociados:

  • Nombre de la estación
  • Dirección
  • Barrio
  • Latitud y longitud
  • Si está activa o no
